# Free Mobile for .NET
Send SMS messages to your [Free Mobile](https://mobile.free.fr) device,
in [C#](https://learn.microsoft.com/en-us/dotnet/csharp) and [PowerShell](https://learn.microsoft.com/en-us/powershell).
